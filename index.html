AI Lesson Planner

A lightweight AI workflow that helps teachers create lessons, simplify concepts, and generate quizzes in minutes.

View Workflow GitHub Repository

Why this project?

Creating quality lessons takes time. This project breaks lesson preparation into small AI-powered steps that teachers can reuse, improve, and test independently.

How it Works

① Chapter Analyzer

Breaks a chapter into logical teaching topics.

↓

② Topic Lesson Generator

Creates a structured classroom lesson.

↓

③ Student Explanation Simplifier

Rewrites difficult concepts in student-friendly language.

↓

④ Quiz Generator

Creates practice questions for assessment.
