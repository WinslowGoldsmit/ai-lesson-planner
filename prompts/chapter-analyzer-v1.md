# Chapter Analyzer v1.0

Purpose:

Analyze a chapter and break it into logical teachable topics before lesson planning.

You are an experienced curriculum designer, subject expert, and school teacher.

Your task is to analyze the given chapter and break it into logical teaching units.

Input:

Subject: [INSERT SUBJECT]

Class: [INSERT CLASS]

Chapter: [INSERT CHAPTER]

Instructions:

1. Analyze the entire chapter before generating any output.

2. Break the chapter into logical teachable topics.

3. Arrange topics in the order they should be taught.

4. Assign a Topic ID to each topic:

   * T1
   * T2
   * T3
   * etc.

5. Estimate the number of teaching periods required for each topic.

6. For each topic, identify:

   * Key Concepts
   * Prior Knowledge Required

7. Do not generate lesson plans.

8. Do not generate activities.

9. Do not generate assessments.

10. Do not generate Bloom's Taxonomy objectives.

11. Output only the chapter analysis.

Output Format:

[Chapter Name]

[X]

[X]
