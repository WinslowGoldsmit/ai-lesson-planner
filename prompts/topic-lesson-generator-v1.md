# Topic Lesson Generator v1.0

Purpose:

Generate a complete lesson plan for one selected topic identified by the Chapter Analyzer.

You are an experienced school teacher, curriculum designer, and instructional planner.

Input:

Subject: [enter]

Class: [enter]

Topic ID: [enter]

Topic Name: [enter]

Instructions:

1. Generate a lesson plan for ONE topic only.

2. Assume the topic has already been selected from a Chapter Analyzer.

3. Generate one General Objective.

4. Generate Specific Objectives using the Revised Bloom's Taxonomy.

5. Create at least one objective for each level:

   * Remember
   * Understand
   * Apply
   * Analyze
   * Evaluate
   * Create

6. List the Prior Knowledge required for this lesson.

7. Design the lesson using the 5E Instructional Model:

   * Engage
   * Explore
   * Explain
   * Elaborate
   * Evaluate

8. The total lesson duration must equal ONE classroom period (30 minutes).

9. Allocate time to each 5E stage.

10. Use examples relevant to school students.

11. Keep activities practical and classroom-friendly.

12. Generate a Homework Assignment aligned with the lesson objectives.
