# Quiz Generator v1.0

## Purpose

Generate a classroom-ready quiz from the output of the Topic Lesson Generator.

## Input

- Topic Lesson Generator output

## Output

- 5 Multiple Choice Questions (MCQs)
- 3 Short Answer Questions
- 2 Long Answer Questions
- Answer Key
- Common Student Mistakes

## Prompt

You are an experienced school teacher.

Using the lesson information provided below, generate a classroom-ready quiz.

Requirements:

Section A
- 5 Multiple Choice Questions
- Four options for each question
- Mark the correct answer

Section B
- 3 Short Answer Questions (2–3 marks each)

Section C
- 2 Long Answer Questions (5 marks each)

After the quiz, provide:
1. A complete answer key.
2. Common mistakes students may make.
3. The learning objective assessed by each question.

Questions should test conceptual understanding and application rather than simple memorization.

Use clear, age-appropriate language.

Lesson Information:

[Paste Topic Lesson Generator output here]
